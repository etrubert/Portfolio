# Instructions pour l'image de fond

Pour que l'image de fond s'affiche correctement sur la page principale :

1. Placez votre image dans ce dossier (`public/`)
2. Nommez-la exactement : `hero-image.jpg`
3. Formats supportés : `.jpg`, `.jpeg`, `.png`, `.webp`

L'image sera automatiquement utilisée comme arrière-plan de la section Hero (page d'accueil).

**Note** : Si votre image a un autre nom ou format, modifiez le fichier `app/globals.css` ligne 72 pour mettre à jour le chemin :
```css
background-image: url('/votre-nom-image.jpg');
```
